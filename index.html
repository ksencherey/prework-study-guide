HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • 3 ways of styling elements: inline, internal and external CSS style sheets
  • margin: this property acts like margin in word processing doc. It indicates how much space we want around the outside of an element
  • padding: adds space around the inside of an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all files and folders in the directory to staging ready for commit
  • git commit -m "Message": commit staged files and folders
  • git push origin branch-name: push commits to github branch

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.JavaScript is an object-oriented programming (OOP) language, and the concepts you're already learning—such as conditional logic and arrays—can be applied to many other in-demand OOP languages.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Using JavaScript, you can power your websites with effects, interactivity, data communication, and much more.
  • The script tag is added to the body element of the HTML file
  • In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • Arrays are zero-indexed and sequential
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A for loop contains the following three important statements: The first statement determines the starting point for our loop. Since we want to start at the beginning of the array, we declare a variable x and give it a value of 0 (because the first index of an array is always 0), as shown in the following line of code: var x = 0;
  • The second statement is the condition. As long as the number in the variable x is less than the length of the array, the loop will keep running. This will allow the loop to execute one time for each item in the array. We use the length built-in array property on our shapes array to express that condition, as shown in the following line of code: x < shapes.length;
  • The final statement is what allows the array to iterate over each item. Since indices follow a predictable pattern, we know the number increases by 1 each time. The code x++ is a shortcut way of writing x+1. Now every time the code loops, x will increase by 1, as shown in the following line of code: x++
  • for(var x = 0; x < shapes.length; x++) { console.log(shapes[x]); }
